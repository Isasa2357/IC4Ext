# IC4Ext v1.0.1

## Added

- HW trigger / SW trigger configuration helpers
- `softwareTrigger()` API for D3D11 / D3D12 camera interfaces
- D3D12 multi-camera synchronized display sample
- Optional recording of the same composed frame that is displayed
- OpenCV multi-camera analysis sample
- Motion-region, mean-luminance, changed-pixel-ratio, and region-count overlays
- Recording of the same analysis-overlaid grid that is displayed
- Public version constants in `IC4Ext/Version.hpp`

## Samples

```text
MultiCameraSyncDisplayD3D12
MultiCameraAnalysisDisplayD3D12
```

The OpenCV analysis sample is optional and is enabled with:

```bat
-DIC4EXT_BUILD_OPENCV_ANALYSIS_SAMPLE:BOOL=ON
-DOpenCV_DIR:PATH=C:\path\to\opencv\build
```

## Validation status

- no-camera tests: expected to run normally
- compile test for the new OpenCV sample: pending user environment
- multi-camera display / HW trigger / SW trigger: requires real cameras
- recording codec availability: depends on the local OpenCV and video codec environment

The `feature/hw-sw-sync` branch must remain unmerged until the real-camera validation is complete.
